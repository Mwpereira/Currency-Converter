# Currency Converter

## 📐 Project Setup

Run the HTML file and that's it!

## 🔐 Back-End

  ### Dependencies
  
  * *Fixer.io* - for grabbing live exchange rates
  * *CountryFlags.io* - for getting country flag images

## 🎨 Front-End

  ### Dependencies
  
  * *Bootstrap* - for UI styling
  * *jQuery* - for HTML DOM manipulation
